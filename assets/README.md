# Assets Directory

This directory contains game assets for Sandlot Sluggers.

## Image Assets
- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon
- `favicon.png` - Web favicon

## Audio Assets
The `sounds/` directory will contain original sound effects:
- Hit sounds (bat connecting with ball)
- Crowd cheers
- Background music
- UI interaction sounds

## Image Assets
The `images/` directory will contain original artwork:
- Character sprites
- Stadium backgrounds
- UI elements

All assets are original creations to avoid copyright issues.
