## Specifications

- **Format:** MP3 (128-192 kbps) or OGG Vorbis
- **Sample Rate:** 44.1 kHz
- **Channels:** Mono for SFX, Stereo for music/ambience
- **Duration:**
  - SFX: 0.1-3 seconds
  - Music: 2-4 minutes (seamless loop)
  - Ambience: 30-60 seconds (seamless loop)
