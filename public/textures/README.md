# Textures Directory

This directory contains all texture files for Sandlot Sluggers.

## Required Textures

### Field Textures
- grass_diffuse.jpg (4096x4096)
- grass_normal.jpg (4096x4096)
- grass_roughness.jpg (2048x2048)
- dirt_diffuse.jpg (2048x2048)
- dirt_normal.jpg (2048x2048)
- dirt_roughness.jpg (2048x2048)
- fence_diffuse.jpg (2048x2048)
- fence_normal.jpg (2048x2048)

### Stadium Skyboxes
Place HDR environment maps here:
- dusty_acres_sky.env or .hdr
- frostbite_field_sky.env or .hdr
- treehouse_park_sky.env or .hdr
- rooftop_rally_sky.env or .hdr
- beach_bash_sky.env or .hdr

Format: HDR cube map, 2048x2048 per face

### UI Textures
- logo.png (512x512, transparent)
- button_normal.png
- button_hover.png
- button_pressed.png

## Specifications

- **Format:** JPG for diffuse, PNG for alpha/masks
- **Color Space:** sRGB for diffuse, Linear for normal/roughness
- **Compression:** Use optimal quality/size balance

## Tools

- **Substance Designer:** Procedural textures
- **Photoshop/GIMP:** Image editing
- **Poly Haven:** Free PBR textures
- **HDRI Haven:** Free HDR skyboxes
